     操作说明

       1.注册账号

         在网站首页注册一个账号，进入个人中心，系统会分配唯一SID作为接口标识

      2.开通接口权限

         联系客服开通接口权限（后续提供自动开通功能），复制页面提供的接口连接，作为超链接放到您的系统页面便可。

    

         



       3.配置域名和回调接口

        配置的业务域名、微信或QQ互联回调接口

        4.开发回调接口

         系统自动完成登录过程、返回登录数据，以url参数（查询字符串即以回调接口url?openid=xx&nickname=XX&headimgurl=XXx&.....）的的形式返回给回调接口。

         包括以下多个参数：

          openid ：微信或者qq分配的用户识别ID

          nickname：用户昵称

         headimgurl：用户头像url，需要进行url解码

          sex：性别

         city：城市

         country：国家

         province：省份



      完成开发。简单便捷一个超链接，一行回调代码便可完成集成工作，不需要多么复杂的框架和集成过程。
